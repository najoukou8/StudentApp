# StudentApp

#Instructions to deploy the project using docker
#prerequires 
- docker installed
- composer installed


### Steps
- clone the project
    ```bash
    git clone https://github.com/najoukou8/StudentApp.git
    cd StudentApp
    ```

- Build the compose
    ```bash
    docker-compose build
    ```
- compose up
    ```bash
    docker-compose up
    ```

- Access the Application
   - Open a web browser and navigate to `http://localhost`.
